## 投資委員会 稟議プロセス

```
投資提案受領
    │
    ▼
┌─────────────────────────────────┐
│  各委員が独立して審査（並列）      │
│                                  │
│  CIO: 戦略整合性 (25点×4項目)    │
│  リスク: リスク評価 (20点×5項目)  │
│  コンプラ: 法令遵守 (25点×4項目)  │
│  クオンツ: 定量評価 (可変配点)    │
└────────────┬────────────────────┘
             │
             ▼
┌─────────────────────────────────┐
│  投票集計                        │
│                                  │
│  賛成率 ≥ 60% && 平均スコア ≥ 50 │
│     → 承認 or 条件付き承認       │
│                                  │
│  コンプライアンス拒否             │
│     → 無条件却下                 │
│                                  │
│  賛成率 < 40%                    │
│     → エスカレーション（人間判断） │
│                                  │
│  それ以外                        │
│     → 却下                       │
└────────────┬────────────────────┘
             │
             ▼
    承認 → リスク最終チェック → 執行
    却下 → ログ記録 → 次回に反映
    エスカレーション → 人間に通知
```

## Agent一覧

| Agent | チーム | 実行タイミング | 主な出力 |
|-------|--------|---------------|---------|
| MarketResearcher | リサーチ | 毎日06:00 | マクロ経済レポート |
| CompanyAnalyst | リサーチ | 毎週月曜+決算期 | 企業分析レポート |
| AlternativeDataAnalyst | リサーチ | 毎日07:00 | オルタナティブシグナル |
| TechnicalAnalyst | 分析 | 場中15分間隔 | テクニカルシグナル |
| QuantitativeAnalyst | 分析 | 毎日08:00 | リスク指標/予測 |
| SentimentAnalyst | 分析 | 毎日08:30 | センチメントスコア |
| StrategyArchitect | 戦略 | 毎日09:00 | 投資提案 |
| InvestmentCommittee | 稟議 | 提案受領時 | 承認/却下決定 |
| RiskManager | 執行 | 承認時 | リスク評価 |
| ExecutionAgent | 執行 | 承認後即時 | 約定レポート |
| PortfolioMonitor | 監視 | 常時 | アラート/日次レポート |
