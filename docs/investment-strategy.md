# 元手50万円から10億円への資産形成戦略

## 概要

本戦略は、初期資金50万円から10億円（2,000倍）の資産形成を目指す長期投資戦略である。
フェーズごとにリスク許容度と手法を変化させ、複利効果を最大化する。

## フェーズ構成

### Phase 1: シード期（50万円 → 500万円）目標期間: 1-2年

**戦略: 集中投資 × モメンタム**

| 項目 | 内容 |
|------|------|
| 手法 | 小型成長株への集中投資（3-5銘柄） |
| 銘柄選定 | 時価総額100億円以下、売上成長率30%以上、営業利益率改善中 |
| エントリー | 出来高急増＋25日移動平均線ブレイク |
| ロスカット | -8%で機械的に損切り |
| 利確 | トレイリングストップ（高値から-15%） |
| レバレッジ | 信用取引は使用しない |
| 再投資率 | 100%（利益は全額再投資） |

**KPI:** 月利10-15%、勝率45%以上、リスクリワード比 1:3以上

### Phase 2: 成長期（500万円 → 5,000万円）目標期間: 2-3年

**戦略: セクターローテーション × イベントドリブン**

| 項目 | 内容 |
|------|------|
| 手法 | 中型株中心（10-15銘柄に分散） |
| 銘柄選定 | セクター分析に基づく有望業種のリーダー銘柄 |
| 追加手法 | 決算プレー、IPOセカンダリー、TOB候補 |
| リスク管理 | 1銘柄あたり総資産の10%以内 |
| ヘッジ | 日経225プットオプションでテールリスクヘッジ |
| 再投資率 | 90%（10%をキャッシュバッファー） |

**KPI:** 年利100-150%、最大ドローダウン-20%以内

### Phase 3: 拡大期（5,000万円 → 5億円）目標期間: 3-5年

**戦略: マルチストラテジー**

| 項目 | 内容 |
|------|------|
| コア（60%） | 中大型グロース株のポートフォリオ |
| サテライト（20%） | イベントドリブン（M&A、スピンオフ、自社株買い） |
| オルタナティブ（10%） | 米国株・ETF（S&P500、NASDAQ100） |
| キャッシュ（10%） | 暴落時の買い増し資金 |
| リスク管理 | セクター集中度30%以下、個別銘柄5%以下 |

**KPI:** 年利50-80%、シャープレシオ1.5以上

### Phase 4: 安定成長期（5億円 → 10億円）目標期間: 2-3年

**戦略: 分散 × 複利 × 配当再投資**

| 項目 | 内容 |
|------|------|
| コア（50%） | 高配当大型株 + インデックスETF |
| グロース（25%） | テーマ型投資（AI、半導体、バイオ） |
| 債券・REIT（15%） | インカムゲイン確保 |
| キャッシュ（10%） | 機動的資金 |
| 目標 | 年利15-25%で安定的に倍増 |

**KPI:** 年利15-25%、最大ドローダウン-10%以内、配当利回り3%以上

## リスク管理の鉄則

1. **ポジションサイジング**: ケリー基準の半分（ハーフケリー）で算出
2. **損切りルール**: フェーズに関わらずエントリー価格の-8%で必ず損切り
3. **ドローダウン制限**: 月次-15%で全ポジション半減、-25%で全決済＋1ヶ月休止
4. **相関管理**: ポートフォリオ内銘柄の相関係数0.5以下を維持
5. **流動性確保**: 1日の出来高の1%以上のポジションは取らない
