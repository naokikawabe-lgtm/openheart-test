# Decisions - 意思決定の記録

## 社長の仕事リスト：フォーカスと責任の明確化

### 2026-03-19（初回設定）

#### テーマ
「社長が毎日見るべき仕事」と「チームに任せること」の線引き

---

#### 【毎日見る！自分の仕事】
1. メンバーにインプットして人の成長を支援し、チームの生産性を底上げすること
2. 既存顧客を担当し、次の大きな提案に繋げていくこと
3. 社長パワーで新規獲得をしまくること
4. 評価制度やHP、カンパニーデックなど、長期で重要になってくることにリソースを割くこと
5. 財務を週次で細かく確認すること

#### 【任せること】
- アサイン調整 → 担当マネージャーに任せる
- 展示会業務（展示会経由の商談、展示会運営） → 展示会担当に任せる
- 採用募集・面談 → 採用チームに任せる

---

#### 運用ルール
- Claudeはこのリストからズレた行動を検知したら指摘する
- 定期的にリマインドする
- 毎週金曜日に「今週の社長の仕事リストとのズレ」を振り返る
