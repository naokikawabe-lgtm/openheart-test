# ブランドガイドライン

## このスキルの目的

株式会社OpenHeartおよびTAVIOブランドの一貫性を維持するためのガイドラインです。すべての対外・対内コミュニケーションにおいて、このガイドラインに準拠した表現を使用してください。

---

## 1. ブランドアイデンティティ

### ミッション/ビジョン
「3Dをもっと身近に、もっと楽しく」

このミッションは、3D技術の敷居を下げ、誰もが3Dコンテンツを作成・活用できる世界を実現するという意志を表しています。

### ブランドパーソナリティ
- **先進的**: 最新のAI技術で3D生成の常識を変える
- **親しみやすい**: 専門家でなくても使える直感的な体験
- **創造的**: ユーザーの想像力を解放する
- **誠実**: 技術の限界も含めて正直に伝える

---

## 2. 名称・表記ルール

### 会社名
- **正式表記**: 株式会社OpenHeart
- **英語表記**: OpenHeart, Inc.
- **NG表記**: オープンハート、openHeart、OPENHEART、Open Heart（スペース入り）

### サービス名
- **正式表記**: TAVIO
- **NG表記**: タビオ、Tavio、tavio、TAVIO（全角英字）
- **説明時の表記**: 3D生成ENGINE「TAVIO」
- **「ENGINE」表記**: TAVIOを説明する文脈では「ENGINE」を大文字で表記する（「エンジン」は本文中の説明時のみ可）

### 補足表記ルール
- 初出時は「3D生成ENGINE『TAVIO』」とフル表記し、2回目以降は「TAVIO」のみで可
- 「3D」は半角英数字+大文字D。「3d」「３D」は不可
- 「AI」は半角大文字。「Ai」「ai」は不可

## 3. トーン & ボイス

### 基本方針
| 場面 | トーン | 文体 |
|------|-------|------|
| 顧客向け資料（提案書・LP） | プロフェッショナルかつ温かみがある | です・ます調 |
| プレスリリース | フォーマルで信頼感がある | です・ます調 |
| SNS・ブログ | カジュアルで親しみやすい | です・ます調（語尾を柔らかく） |
| 社内文書 | 簡潔で明確 | である調も可 |
| 投資家向け資料 | 論理的でデータドリブン | です・ます調 |

### トーンの具体例

**良い例:**
- 「TAVIOを使えば、テキストから3Dモデルをスムーズに生成できます」
- 「3Dコンテンツ制作のハードルを、テクノロジーの力で下げていきます」
- 「直感的な操作で、あなたのアイデアを3Dの世界に」

**悪い例:**
- 「TAVIOなら誰でも簡単に3Dモデルが作れます」（過度に安易さを強調）
- 「競合のA社より圧倒的に優れています」（競合批判）
- 「業界最高の3D生成技術」（根拠のない最上級表現）

## 4. 禁止表現一覧

| カテゴリ | 禁止表現 | 代替表現 |
|---------|---------|---------|
| 過度な簡易さ | 簡単、誰でもできる | 直感的、スムーズ、手軽に |
| 根拠なし最上級 | 業界No.1、最高、最強 | 先進的、高品質、高精度 |
| 絶対的表現 | 絶対、完璧、100% | 高い精度で、信頼性の高い |
| 競合批判 | A社より優れている | （自社の強みのみを述べる） |
| 曖昧な技術説明 | AIが勝手に作る | AIが入力を解析し生成する |
| 過度な約束 | 必ず成功する | 効果的にサポートする |

## 5. ビジュアルガイドライン

### カラーパレット
- **プライマリカラー**: ブランド指定色（ロゴカラーに準拠）
- **セカンダリカラー**: アクセントとして使用
- **背景色**: 白または薄いグレーを基本とする
- **テキスト色**: ダークグレー（#333333）を基本

### フォント指針
- **見出し**: ゴシック体系（Noto Sans JP、ヒラギノ角ゴ等）
- **本文**: ゴシック体系を基本（明朝体は特別な演出がある場合のみ）
- **英字**: Inter、Helvetica Neue等のサンセリフ体

### 画像・図版
- TAVIOで生成した3Dモデルの画像を積極的に活用する
- ストックフォトよりもオリジナル素材を優先する
- 暗すぎる・重すぎるビジュアルは避け、明るく開放的な印象を保つ

## 6. チェックリスト

対外向け資料を公開・送付する前に、以下を確認すること:

- [ ] 会社名が「株式会社OpenHeart」と正しく表記されているか
- [ ] サービス名が「TAVIO」と正しく表記されているか
- [ ] 禁止表現が含まれていないか
- [ ] トーンが対象読者に適切か
- [ ] ミッション引用時に「3Dをもっと身近に、もっと楽しく」が正確か
- [ ] 技術的な記述に誤りがないか
- [ ] 競合他社を名指しで批判していないか
- [ ] 根拠のない数値や主張がないか
- [ ] ビジュアルがブランドカラーに準拠しているか

## 7. よくあるミス

1. **「Tavio」と表記してしまう** → 正しくは「TAVIO」（全大文字）
2. **「オープンハート」とカタカナ表記** → 正しくは「OpenHeart」
3. **「簡単に3Dが作れる」と書いてしまう** → 「直感的に3Dモデルを生成できる」
4. **ミッションを不正確に引用** → 「3Dをもっと身近に、もっと楽しく」を正確に
5. **「3d」と小文字のdを使用** → 「3D」が正しい
6. **「AI エンジン」と表記** → 「3D生成ENGINE」が正式表記
