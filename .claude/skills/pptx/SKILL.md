# PPTX生成スキル

## いつ使うか
- ユーザーが提案資料・プレゼンスライドの作成を依頼したとき
- Markdownの構成案からPPTXを生成する必要があるとき
- 既存のPPTXテンプレートを編集して新しいスライドを作るとき

## 何をするか
Markdownで整理された構成案をもとに、PPTXファイルを生成・編集する。
提案資料の「考える作業」と「整形する作業」を分離し、品質の高いスライドを効率的に作成する。

## 何ができるか
- Markdownの構成案からPPTXのXMLを生成
- テンプレートPPTXのXMLを編集してカスタマイズ
- サブエージェントによるビジュアルQA（テキスト切れ・レイアウト崩れの検出と修正）
- 生成→QA→修正のサイクルをGitで記録

## ワークフロー
1. **Markdownで構成を固める** — ストーリーライン、各スライドの主張、根拠を整理
2. **Plan Modeでレイアウトを選定** — 伝えたい内容の型からレイアウトパターンを逆引き
3. **XMLを編集してPPTX生成** — テンプレートのXMLを直接編集
4. **ビジュアルQA** — PDF/画像に変換してサブエージェントで検証・修正

## 関連ファイル
- `editing.md` — テンプレートからの編集手順
- `templates/proposal.md` — 提案資料のMarkdownテンプレート
- `deliverables/` — 生成された成果物の保存先
