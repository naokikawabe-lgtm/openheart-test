# バカラ戦略ガイド: 20万円 → 50万円

## 用語集

| 用語 | 意味 |
|------|------|
| **バンカー (Banker)** | バカラの2つの賭け先の一つ。「胴元側」という意味だが、実際にはただの選択肢。ルール上3枚目を引く条件がわずかに有利なため、勝率が若干高い (約45.86%)。勝つと配当から5%の手数料が引かれる |
| **ハウスエッジ (House Edge)** | カジノ側の取り分の割合。たとえばハウスエッジ1%なら、長期的に100円賭けるごとに平均1円がカジノの利益になる。数値が低いほどプレイヤーに有利。バカラのバンカーベットは1.06%で、カジノゲームの中でもトップクラスに低い |
| **タイ (Tie)** | バンカーとプレイヤーの点数が同じ「引き分け」のこと。配当8倍と魅力的に見えるが、ハウスエッジが14.36%と非常に高く、割に合わない賭け。出現率は約9.5% |
| **バンクロール (Bankroll)** | ギャンブルに使う総資金のこと。今回の場合は元手の20万円。バンクロール管理 = 資金をどう配分して賭けるかの管理術。一般的に1回のベットはバンクロールの5%以下が安全とされる |
| **フラットベット (Flat Bet)** | 毎回同じ金額を賭け続ける方法。例: 毎ハンド1万円ずつ賭ける。勝っても負けても金額を変えない。派手さはないが、資金の減りが緩やかで最もリスクが低い賭け方 |

## バカラの戦略をやさしく解説

### そもそもバカラって何をするゲーム？

バカラはとてもシンプルなカードゲームです。やることは **「AとBのどちらが勝つかを予想する」だけ** です。

テーブルには「A側 (プレイヤー側)」と「B側 (バンカー側)」があり、ディーラーがそれぞれにカードを配ります。カードの合計点が **9に近い方が勝ち** です。あなたは自分でカードを触る必要はなく、ただ「Aが勝つ」「Bが勝つ」「引き分け」のどれかにチップを置くだけです。

**運動会の赤白の玉入れで「赤が勝つ」に賭けるようなもの** だと思ってください。

---

### 戦略のポイントをやさしく説明

#### ポイント1: 毎回「B側 (バンカー側)」に賭け続ける

A側とB側では、ルールの仕組み上 **B側がほんの少しだけ勝ちやすい** です (勝率 約45.9% vs 44.6%)。この差は小さいですが、何十回、何百回と繰り返すと差がつきます。

> **例**: 100回勝負したとすると
> - B側に賭け続けた場合 → 約46回勝ち、約44回負け、約10回引き分け
> - A側に賭け続けた場合 → 約45回勝ち、約45回負け、約10回引き分け
>
> たった1回の差ですが、これが積み重なると収支に大きく影響します。

**注意**: B側で勝った場合、配当から5%がカジノに引かれます。1万円賭けて勝つと、手元に来るのは19,500円 (9,500円の利益) です。それでもA側より有利です。

#### ポイント2: 「引き分け」には絶対に賭けない

引き分けは配当が8倍と高く魅力的に見えますが、**起きる確率が約9.5%しかない** のに対し、配当が割に合いません。

> **例**: わかりやすく言うと
> - 10回に1回しか当たらないクジを、1万円で買い続ける
> - 当たっても8万円しかもらえない (10万円もらえてやっとトントン)
> - 100回買うと → 100万円使って、約80万円しか戻ってこない
>
> これが「引き分けはカジノが儲かる罠」と言われる理由です。

#### ポイント3: 毎回同じ金額を賭ける

「負けたら倍額を賭けて取り返す」という方法は一見よさそうですが、**連敗すると一瞬で資金が尽きます**。

> **例**: 1万円スタートで負けるたびに倍にする場合
> - 1回目: 1万円負け (累計 -1万)
> - 2回目: 2万円負け (累計 -3万)
> - 3回目: 4万円負け (累計 -7万)
> - 4回目: 8万円負け (累計 -15万)
> - 5回目: 次に16万円必要だが、残り5万円で賭けられない → **ゲームオーバー**
>
> たった5連敗で20万円のうち15万円が消えます。5連敗は珍しくなく、約3%の確率 (30回に1回程度) で起きます。

毎回同じ1万円を賭け続ければ、20連敗しても手元に0円にはなりません。**地味ですが一番安全**です。

#### ポイント4: 手持ちが増えたら、段階的に賭け金を上げる

最初は1万円ずつ。30万円まで増えたら1.5万円、40万円まで増えたら2万円に引き上げます。
資金が増えた分だけ少し大きく賭け、**減ったら元の額に戻す**。

> **例**: 実際の流れのイメージ
>
> ```
> 【第1段階】手持ち20万円 → 1万円ずつ賭ける
>   1回目: B側に1万 → 勝ち → 手持ち 20万9500円
>   2回目: B側に1万 → 負け → 手持ち 19万9500円
>   3回目: B側に1万 → 勝ち → 手持ち 20万9000円
>   ... (繰り返し)
>   → 運良く手持ちが30万円を超えた！
>
> 【第2段階】手持ち30万円 → 1.5万円ずつに引き上げ
>   → 手持ちが25万円に減ってしまったら、1万円に戻す
>   → 運良く40万円を超えた！
>
> 【第3段階】手持ち40万円 → 2万円ずつに引き上げ
>   → 手持ちが35万円に減ったら、1.5万円に戻す
>   → 50万円到達！ → すぐにテーブルを離れる！
> ```

#### ポイント5: やめるタイミングを先に決めておく

これが **最も重要** です。

- **勝ちの場合**: 50万円に達したら、その瞬間に席を立つ。「もうちょっと」は禁物
- **負けの場合**: 手持ちが10万円まで減ったら撤退する (10万円は持って帰る)

> **例**: なぜこれが大事か
> - Aさん: 30万円まで増えた → 「もっといける」 → 結局18万円まで減って終了
> - Bさん: 30万円まで増えた → 50万でやめると決めていた → 結果的に50万到達で離席
> - Cさん: 50万到達 → 「100万いけるかも」 → 結局15万円まで減って終了
>
> **カジノで一番多い負けパターンは「やめ時を見失うこと」** です。

---

### 3行まとめ

1. **B側 (バンカー) だけに賭ける** — それだけでカジノゲームの中で最も有利な選択になる
2. **毎回同じ金額を賭ける** — 資金が増えたら少し上げ、減ったら戻す
3. **50万で即やめ、10万で撤退** — 何があってもこのルールを破らない

## カジノゲーム比較: 20万→50万の達成確率

### ハウスエッジ比較

| ゲーム | ハウスエッジ | 補足 |
|--------|-------------|------|
| **ブラックジャック (ベーシックストラテジー)** | **約0.5%** | 最小。ただしルール暗記が必須 |
| バカラ (バンカー) | 1.06% | シンプルで初心者向き |
| クラップス (Don't Pass + Odds) | 0.37〜0.47% | オッズベット併用時。韓国ではテーブルが少ない |
| ヨーロピアンルーレット (赤黒/偶奇) | 2.70% | 非推奨。ハウスエッジが高い |
| スロット | 2〜15% | 論外 |

### 達成確率の比較 (フラットベット、1万円/回)

| ゲーム | 20万→50万の達成確率 | 理由 |
|--------|---------------------|------|
| **ブラックジャック (BS完璧)** | **約38〜42%** | ハウスエッジ最小のため最も長く戦える |
| バカラ (バンカー) | 約35〜40% | 次点。操作が簡単で安定 |
| クラップス (Don't Pass + Max Odds) | 約37〜41% | オッズベット (ハウスエッジ0%) が強力 |
| ルーレット (赤黒) | 約30〜34% | ハウスエッジが大きく不利 |

### 結論: ブラックジャック vs バカラ

**数学的に最も有利なのはブラックジャック** (ベーシックストラテジー完璧実行時)。

ただし現実的には:

| 観点 | ブラックジャック | バカラ |
|------|-----------------|--------|
| ハウスエッジ | **0.5% (最良)** | 1.06% |
| 難易度 | 高い (数百パターンの判断表を暗記) | **超簡単 (バンカーに賭けるだけ)** |
| プレイ速度 | 遅い (判断に時間がかかる) | **速い** |
| ミスのリスク | 判断ミスでハウスエッジが2〜3%に悪化 | **ミスしようがない** |
| 精神的負荷 | 高い (毎回判断が必要) | **低い** |
| テーブル数 | 韓国カジノでは少なめ | **韓国カジノの主力** |

**おすすめ**:
- ベーシックストラテジーを完璧に暗記できる → **ブラックジャック** (達成確率 最大約42%)
- 暗記は面倒 / 気楽にやりたい → **バカラ** (達成確率 約35〜40%)
- 中途半端にブラックジャックをやるくらいなら、バカラの方が確実に有利

> **重要**: どのゲームを選んでも達成確率は最大約40%。つまり **10回挑戦して4回成功、6回失敗** が現実です。カジノに「高確率で勝てるゲーム」は存在しません。

## 推奨戦略: 段階的利確 + フラットベット併用

### Phase 1: 安定フェーズ (20万 → 30万)

- **1ベット額**: 10,000円 (バンクロールの5%)
- **賭け先**: バンカー固定
- **目標**: +100,000円 (約105ハンドの期待プレイ数)
- 調子が悪ければ **50,000円負けた時点 (15万円) で撤退判断**

### Phase 2: 攻めフェーズ (30万 → 40万)

- **1ベット額**: 15,000円に引き上げ
- **賭け先**: バンカー固定
- **損切りライン**: 25万円まで減ったらPhase 1に戻る

### Phase 3: 仕上げフェーズ (40万 → 50万)

- **1ベット額**: 20,000円
- **賭け先**: バンカー固定
- **35万円まで減ったらPhase 2に戻る**
- **50万円到達で即離席**

## ベッティングシステム比較

| システム | リスク | 特徴 |
|---------|--------|------|
| **フラットベット** (推奨) | 低 | 毎回同額。安定。長く遊べる |
| マーチンゲール | 極高 | 負けたら倍賭け。連敗で即破産 |
| パーレー (逆マーチン) | 中 | 勝ったら倍賭け。3連勝で利確 |
| 1-3-2-6法 | 中低 | 連勝時に利益を伸ばす。損失限定的 |

### パーレー法 (短期勝負向き)

3連勝で大きく稼ぎたい場合:

1. 10,000円 → 勝ち → 20,000円に
2. 20,000円 → 勝ち → 40,000円に
3. 40,000円 → 勝ち → **70,000円の利益** (コミッション引後)
4. 3連勝したら最初に戻す

- 3連勝確率: 約12.5%
- 失敗しても1回分 (10,000円) の損失

### 1-3-2-6法 (バランス型)

ベース10,000円の場合:

1. 10,000円賭け → 勝ち
2. 30,000円賭け → 勝ち
3. 20,000円賭け → 勝ち
4. 60,000円賭け → 勝ち → **合計利益: 約114,000円**

- どのステップで負けても最大損失は限定的
- ステップ2で負けても -20,000円

## 絶対守るべきルール

1. **タイ (引き分け) には絶対賭けない** — ハウスエッジ14.36%
2. **50万円到達で即やめる** — 欲を出さない
3. **損切りライン厳守** — 10万円 (元手の50%) 失ったら撤退
4. **酒を飲みながらやらない** — 判断力が鈍る
5. **罫線 (出目表) を過信しない** — 各ハンドは独立事象
6. **連敗時に賭け金を上げない** — 冷静さを失うサイン

## 現実的な期待値の話

バカラはカジノゲームの中で最もハウスエッジが低いゲームの一つですが、 **長期的にはカジノが必ず勝つ** 構造です。

- バンカーベットの期待値: 賭け金 × 0.9894 (1ハンドあたり約1%の損失)
- 200,000円を全てバンカーに賭けた場合の期待損失: 約2,120円

**20万→50万は「2.5倍にする」ということ。これは短期の運次第であり、戦略で保証できるものではありません。**

達成確率の概算 (フラットベット、バンカー固定):
- 1回あたり1万円ベットの場合: 約 **35〜40%** で50万到達 (破産前に)
- ただし達成前に10万円の損切りに引っかかる確率: 約 **60〜65%**

## 韓国カジノの注意点

- **パスポート必須** (外国人専用カジノ)
- **ミニマムベット**: テーブルにより異なる (一般的に10,000〜50,000ウォン)
- **両替**: カジノ内の両替所が便利 (レートはやや不利)
- **主要カジノ**: パラダイスシティ (仁川)、セブンラックカジノ (ソウル)、ウォーカーヒルカジノ (ソウル)
- **チップ**: ディーラーへのチップは任意だがマナーとして推奨
