# バカラ戦略ガイド: 20万円 → 50万円

## バカラとは？

カードゲームの一種で、やることは **「AとBのどちらが勝つかを予想する」だけ**。テーブルの「A側」と「B側」にそれぞれカードが配られ、合計点が **9に近い方が勝ち**。あなたはカードに触れず、「A勝ち」「B勝ち」「引き分け」のどれかにチップを置くだけです。

## 用語集

| 用語 | 意味 |
|------|------|
| **A側 / B側** | テーブル上の2つの賭け先 (正式にはプレイヤー / バンカー)。B側がルール上わずかに有利 |
| **カジノの取り分** | カジノ側が長期的に得る利益率。バカラのB側は約1%で、カジノゲームの中でもトップクラスに低い |
| **引き分け** | A側とB側が同点。配当8倍だがカジノの取り分が約14%と非常に高い罠 |
| **元手** | ギャンブルに使う総資金。1回の賭けは元手の5%以下が安全 |
| **定額賭け** | 毎回同じ金額を賭ける方法。最もリスクが低い |

## 前提

- **元手**: 20万円 → **目標**: 50万円 (2.5倍)
- 達成確率: 約 **35〜40%** (定額1万円、B側固定の場合)
- 達成前に損切りになる確率: 約 **60〜65%**

> **現実**: どんな戦略でも達成確率は最大約40%。10回挑戦して4回成功、6回失敗。カジノに「高確率で勝てるゲーム」は存在しません。

## なぜバカラか？ — 他ゲームとの比較

| ゲーム | カジノの取り分 | 達成確率 | 補足 |
|--------|---------------|---------|------|
| ブラックジャック | 約0.5% | 約38〜42% | 最良だがルール暗記が必須。判断ミスで大幅悪化 |
| **バカラ (B側)** | **1.06%** | **約35〜40%** | **超簡単。B側に置くだけ** |
| ルーレット | 2.70% | 約30〜34% | 取り分が大きく不利 |
| スロット | 2〜15% | — | 論外 |

**結論**: ルール暗記が得意ならブラックジャック、そうでなければバカラが最適。中途半端にブラックジャックをやるくらいならバカラの方が確実に有利。

## 5つの鉄則

### 1. 毎回B側に賭ける

B側はルールの仕組み上、わずかに勝ちやすい (勝率 約45.9% vs 44.6%)。

> **例**: 100回勝負すると
> - B側 → 約46勝44敗10分け
> - A側 → 約45勝45敗10分け
>
> たった1回の差だが、積み重なると収支に大きく効く。
> B側は勝利時に5%引かれるが (1万円賭けて勝つと利益9,500円)、それでもA側より有利。

### 2. 引き分けには絶対賭けない

配当8倍に見えるが、確率が約9.5%なので割に合わない。

> **例**: 1万円ずつ100回賭けると → 100万円使って約80万円しか戻らない (カジノが約14%取る)

### 3. 毎回同じ金額を賭ける

「負けたら倍賭け」は危険。

> **例**: 1万円スタートで倍々にすると
> 1回目 -1万 → 2回目 -3万 → 3回目 -7万 → 4回目 -15万 → 5回目に16万必要だが残り5万 → **破産**
>
> 5連敗は約3%の確率 (30回に1回) で起きる。毎回1万円固定なら20連敗しても資金は残る。

### 4. 段階的に賭け金を上げる

| 段階 | 手持ち | 1回の賭け | 撤退ライン |
|------|--------|----------|-----------|
| 第1段階 | 20万〜30万 | 1万円 | 15万で撤退判断 |
| 第2段階 | 30万〜40万 | 1.5万円 | 25万で第1段階に戻る |
| 第3段階 | 40万〜50万 | 2万円 | 35万で第2段階に戻る |

> **例**: 流れのイメージ
> ```
> 手持ち20万 → 1万ずつ → 勝ち負けを繰り返し → 30万突破！
> → 1.5万に引き上げ → 25万に減ったら1万に戻す → 40万突破！
> → 2万に引き上げ → 50万到達 → 即離席！
> ```

### 5. やめるタイミングを先に決めておく

- **利確**: 50万円で即離席。「もうちょっと」は禁物
- **損切り**: 10万円まで減ったら撤退

> **例**:
> - Aさん: 30万まで増加 → 欲を出す → 18万で終了
> - Bさん: 50万到達 → 「100万いけるかも」 → 15万で終了
>
> **カジノで一番多い負けパターンは「やめ時を見失うこと」**

## 賭け方のバリエーション

| 方式 | リスク | 特徴 |
|------|--------|------|
| **定額賭け** (推奨) | 低 | 毎回同額。安定。長く遊べる |
| 倍賭け | 極高 | 負けたら倍。連敗で即破産 |
| 逆倍賭け | 中 | 勝ったら倍。3連勝で利確 |
| 1-3-2-6法 | 中低 | 連勝時に利益を伸ばす。損失限定的 |

### 逆倍賭け (短期勝負向き)

1万円 → 勝ち → 2万円 → 勝ち → 4万円 → 勝ち → **約7万円の利益**。3連勝したら最初に戻す。失敗しても1万円の損失。3連勝確率は約12.5%。

### 1-3-2-6法 (バランス型)

1万 → 勝ち → 3万 → 勝ち → 2万 → 勝ち → 6万 → 勝ち → **約11.4万円の利益**。どの段階で負けても最大損失は限定的。

## 絶対守るべきルール

1. **引き分けには絶対賭けない**
2. **50万円で即やめる** — 欲を出さない
3. **10万円失ったら撤退** — 損切り厳守
4. **酒を飲みながらやらない**
5. **過去の勝敗パターンを過信しない** — 毎回の勝負は独立した事象
6. **連敗時に賭け金を上げない** — 冷静さを失うサイン

## 韓国カジノの注意点

- パスポート必須 (外国人専用)
- 最低賭け金はテーブルにより異なる (一般的に1〜5万ウォン)
- 主要カジノ: パラダイスシティ (仁川)、セブンラックカジノ (ソウル)、ウォーカーヒルカジノ (ソウル)
